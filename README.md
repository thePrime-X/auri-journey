## 🌌 Auri’s Journey

**Auri’s Journey** is a mobile puzzle-based learning game that introduces the fundamentals of computational thinking through interactive gameplay.

Players guide **Auri**, a small robot, by building simple command sequences to solve grid-based challenges. Instead of focusing on coding syntax, the game emphasizes core concepts such as:

- logical sequencing  
- problem decomposition  
- spatial reasoning  
- algorithmic thinking
